# Upcoming event images

Place your event/location images here. They will be served at `/images/upcoming/your-file.jpg`.

**How to use:**
1. Add image files (e.g. `las-vegas.jpg`, `chicago.jpg`, `san-francisco.jpg`).
2. In `components/UpcomingSchedule.tsx`, set each locationâ€™s `imageUrl` to your file, for example:
   - `imageUrl: "/images/upcoming/las-vegas.jpg"`

**Tips:**
- Use descriptive names: `las-vegas-2026.jpg`, `chicago-mixer.jpg`, etc.
- JPG, PNG, or WebP are fine. Keep file sizes reasonable (e.g. under 500KB) for faster loading.
